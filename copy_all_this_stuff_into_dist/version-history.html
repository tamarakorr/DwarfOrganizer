Revision Details

v1.11

Functionality

  • Default job settings should be more sensible

Problems resolved

  • Fixed a crash if last dwarves.XML not found
  • Fixed a crash with non-numeric input in some Job Settings columns
  • The DEFAULT SETTINGS and MY DEFAULT SETTINGS files for Job Settings now work as intended

Interface

  • Dwarf List resizing improved
  • Added Nickname column (optional) to optimizer results
  • More job filters for optimizer results
  • Job Settings table:
    • Copy and paste now supported
    • Ctrl, Alt, and Meta keys no longer initiate cell editing
  • Default focus to File Name in open/save dialogs (Nimbus bug workaround)

v1.1

Functionality

  • The job blacklist/whitelist can now be displayed and edited using Window->Rules Editor. The tutorial has been updated to reflect these changes.
  • The tutorial is now also accessible using the menu (Help->Tutorial). It is opened in the default web browser.
  • Slight change to job blacklist file structure: comments are now added at the end of a line. (Old files still work and are updated automatically at next save.)

Problems resolved

  • Fixed an issue that was breaking the job whitelist
  • Fixed a problem that was sometimes breaking the Dwarf List Columns menu

Interface

  • Added a "Window" menu that can be used to bring windows to the front
  • Organized menus a little better

v1.1 Outstanding issues

  • There are a couple small resizing bugs in Nimbus, the Java look-and-feel manager I'm using. This affects some parts of the Edit Rules screen. Users might have to enlarge the window a little to read all the labels. I am looking into permanent solutions.
  • I should have mentioned this with the original release--Dwarf Organizer doesn't know the stats associated with the following skills:
    • Wool Spinning
    • Press Operation
    • Wax Working
    • Bee Keeping
    • Glazing
    • Pottery
    • Shearing

    In these cases, I attempted to follow the pattern. The stats were copied from labors that seemed very similar.

    This is noted in labor-skills.XML and list-explanations.txt. But normal use of Dwarf Organizer probably wouldn't bring this to notice.

    If I find out the stats associated with any of these labors, I'll update Dwarf Organizer with the new information. Until then, please take everything Dwarf Organizer says about those particular labors with a grain of salt!
