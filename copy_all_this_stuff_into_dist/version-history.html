Revision Details

v1.3

New Feature: View Manager

  • The current column ordering in the Dwarf List can be saved and given a name using View->Save Current View As...
  • Views can be renamed and deleted using View->View Manager
  • Switch between saved views with the View menu

Dwarf List improvements

  • Columns can be frozen by dragging the new divider from the left edge of the table
  • The Columns menu provides finer control over which columns are displayed

Other improvements

  • New "config" directory structure: "config/default" contains installation defaults. User files are still saved to "/config". When reading data files, the application always checks for a user file first, and then resorts to a default file if not found.
  • Menus in the active internal window are combined into the main menu
  • Improved error handling for failure to read files

Problems fixed

  • Columns->Hide All now lives up to its name, hiding all except the two required columns
  • When the main window is closing, the user is always prompted to save any unsaved changes (except Job Settings)

v1.21

Minor General Improvements

  • New Tutorial section explains calculation methods
  • The Help->Tutorial menu should now work on operating systems not implementing Java 6 Desktop
  • Users are now prompted about unsaved changes to rules/exclusions when closing the main window
  • Minor loading speed enhancements

Minor Interface improvements

  • [About] Displays the software version now
  • [Rules Editor] Most interface annoyances defeated
  • [Dwarf List] Added Show All and Hide All options to Columns menu
  • [Dwarf List] Added optional column formatting for stats, skill potentials, and skill levels (values; text; values and text)
  • [Dwarf List] Improvements when copying rows to clipboard: Column headings are now copied; line breaks and tabs convert to spaces
  • [Dwarf List] Added basic view framework and an optional Military view (feature incomplete)
  • [Job List] Cell editing now cancels/stops properly when loading/saving

v1.20

New Feature

  • Added the Exclusion Manager, for remembering groups of dwarves to exclude. The tutorial has been updated with a new section.

Problem Resolved

  • [Dwarf List] The Columns menu works once more!

Minor Improvements

  • [Rules Editor] Fixed the Nimbus layout overlapping. (Everything should be readable without resizing the window.)
  • [Dwarf List] More columns can be shown/hidden.
  • [Dwarf List] The column preferences and layout no longer reset when loading a new file. (Preferences aren't remembered between sessions...yet.)

v1.14

  • Reverted the bad Java 7 things I did. Primitives as column classes changed back to objects.

v1.1

Functionality

  • The job blacklist/whitelist can now be displayed and edited using Window->Rules Editor. The tutorial has been updated to reflect these changes.
  • The tutorial is now also accessible using the menu (Help->Tutorial). It is opened in the default web browser.
  • Slight change to job blacklist file structure: comments are now added at the end of a line. (Old files still work and are updated automatically at next save.)

Problems resolved

  • Fixed an issue that was breaking the job whitelist
  • Fixed a problem that was sometimes breaking the Dwarf List Columns menu

Interface

  • Added a "Window" menu that can be used to bring windows to the front
  • Organized menus a little better

v1.1 Outstanding issues

  • There are a couple small resizing bugs in Nimbus, the Java look-and-feel manager I'm using. This affects some parts of the Edit Rules screen. Users might have to enlarge the window a little to read all the labels. I am looking into permanent solutions.
  • I should have mentioned this with the original release--Dwarf Organizer doesn't know the stats associated with the following skills:
    • Wool Spinning
    • Press Operation
    • Wax Working
    • Bee Keeping
    • Glazing
    • Pottery
    • Shearing

    In these cases, I attempted to follow the pattern. The stats were copied from labors that seemed very similar.

    This is noted in labor-skills.XML and list-explanations.txt. But normal use of Dwarf Organizer probably wouldn't bring this to notice.

    If I find out the stats associated with any of these labors, I'll update Dwarf Organizer with the new information. Until then, please take everything Dwarf Organizer says about those particular labors with a grain of salt!
